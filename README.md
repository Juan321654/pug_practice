```npm i pug-cli -g```

#### watch files in the current directory
```pug -w ./```

#### output to folder
```-o ./html```

#### prettyfy the output 
```-P```

#### full terminal command
```pug -w ./ -o ./html -P```


#### watch specific file 
```pug -w extra.pug -o ./html -P```
